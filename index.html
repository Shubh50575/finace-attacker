Welcome to the Finance Tracker

This is a simple Python project to keep track of your income and expenses. You can add transactions, view them, and get a summary of your financial balance.

How to Run:

  1. Make sure you have Python installed on your system.
  2. Save the project code in a file named finance_tracker.py.
  3. Open a terminal in the project folder.
  4. Run the command: python finance_tracker.py
